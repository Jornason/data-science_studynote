数据科学学习笔记（use R）

前言

这是参考Johns Hopkins在coursera上开设的系列课程data science而作的学习笔记。并不承诺包涵系列课程的全部内容，也不承诺所有内容均忠实于原课程。只能说笔记主要内容是对data science 9门课的课件，quiz，peer assignment所涉及的大部分知识，并在不违背课程荣誉准则与data science课程著作权利的情形下，加以总结整理。还是那句老话，阅读原著有利身心健康，二手资料难免消化不良。

本笔记主要将9门课程分散为三个部分的内容：

  • Informatics
    • data import and export
    • data cleaning
  • Analytic
    • exploratory data stucture
    • statistical estimation and prediction
    • machine learning
  • Visualization
    • plotting
    • developing data products

主要内容介绍如下：

  • 数据信息部分分为数据的导入与导出和数据清洗两个部分。数据的导入与导出主要介绍各种数据类型如Excel，Jason，XML如何导入R中，以及如何获取网络数据，如何连接MySQL等内容。数据清洗则主要介绍数据的管理，操作，综合。

  • 数据分析部分分为探索性数据分析，统计推断，机器学习三个部分。探索性数据分析主要集中于介绍聚类的相关知识，统计推断部分主要介绍数理统计中估计与检验的最基础的知识，因为过于简单，不会占用过多的篇幅。机器学习部分会介绍分类与回归两种常用的data mining的办法，预期能够涵盖数据挖掘十大经典算法。

  • 数据可视化部分分为作图与数据产品的开发两个部分。作图主要介绍R中画图常用的lattice与ggplot两个系统。数据产品开发集中介绍shinny与slidfy两个包的使用，更多内容诸如开发R package，rCharts等敬请关注coursera的课程内容，这里不再赘述。
